# HESIA Open Engineering Staging

This staging folder prepares the open-source material explicitly mentioned in the marketing plan.

It contains three GitHub-ready repositories:

- `repositories/hesia-pqc-bench`
- `repositories/optee-ta-skeleton`
- `repositories/pqc-migration-guide`

Important guardrail:

- the proprietary HESIA core product is intentionally excluded
- no production keys, signatures, sealed blobs, or customer-facing runtime code are copied here
- the OP-TEE material is provided as a generic public skeleton, not as a dump of the internal HESIA TA

Use this folder as a publication workspace: each subfolder can be extracted into its own GitHub repository.
