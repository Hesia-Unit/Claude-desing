# HESIA — Dossier Marketing & Go-to-Market

Ce dossier contient la **campagne marketing complète** de HESIA : étude de marché, plan stratégique, identité de marque, site web, campagnes publicitaires, contenus, pipeline commercial, KPI, budget et relations presse.

## Contexte

HESIA est une plateforme logicielle de sécurisation de drones et systèmes autonomes embarqués :
- Firmware durci avec cryptographie post-quantique (ML-KEM-1024, ML-DSA-87).
- OP-TEE Trusted Application pour attestation et sealing matériel.
- Serveur de contrôle avec TLS 1.3 mTLS, policy signée, allowlist Ed25519.
- Pipeline perception IA embarquée (YOLO + MiDaS).

Cible stratégique retenue dans ce dossier :
1. **Phase 1 (T+0 à T+18 mois)** : dual-use civil — inspection industrielle, infrastructures critiques (énergie, transport, télécoms).
2. **Phase 2 (T+18 à T+36 mois)** : défense européenne souveraine, OIV, dual-use militaire.

## Hypothèses méthodologiques

Cette campagne est construite sans accès à :
- Chiffres financiers internes HESIA.
- Études de marché propriétaires (Gartner, IDC, Frost & Sullivan).
- Contrats / LOI clients existants.

Les chiffres fournis sont des **ordres de grandeur sourcés** depuis des études publiques (Teal Group, PwC, DSA Group, ANSSI, Commission européenne), et doivent être **validés** par un consultant marché si l'entreprise engage des investissements significatifs.

## Philosophie du message

Trois principes directeurs adoptés partout dans ce dossier :

1. **Crédibilité avant hype** : la cible (CSO / CTO / responsables innovation dans l'énergie, défense, sécurité) détecte le marketing bullshit. Arguments techniques précis, pas de buzzwords creux.
2. **Souveraineté assumée** : positionnement européen affirmé (RGPD, NIS2, CRA, ANSSI), sans agressivité anti-US/anti-Chine.
3. **Roadmap honnête** : HESIA est au TRL 4-5 aujourd'hui (cf. audit). On communique sur la trajectoire et les validations à venir, pas sur un produit parfait.

## Séquençage suggéré

- **Mois 0-3** : validation étude de marché + identité de marque + site web + LinkedIn.
- **Mois 3-6** : contenus techniques (livre blanc PQC, blog posts), premiers webinaires, outreach.
- **Mois 6-12** : campagnes payantes, présence salon (Milipol, Eurosatory pour phase 2), PR.
- **Mois 12+** : case studies clients, tournée conférences techniques, rapport d'impact.

Tous les documents sont conçus pour être **lus et exécutés** par un CMO / responsable marketing seul ou avec une équipe réduite (2-3 personnes).
